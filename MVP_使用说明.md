# Hand+Music 2x2 MVP 版本使用说明

## 📦 项目文件结构

```
hand+music 2x2/
├── index.html              # 主 HTML 文件
├── main.js                 # 主应用入口
├── HandTracker.js          # 手势识别模块
├── EuclideanEngine.js      # 欧几里得算法引擎
├── MusicManager.js         # 音乐管理器（包含音色库）
├── VisualRenderer.js       # 可视化渲染器
├── README.md               # 项目说明文档
└── MVP_使用说明.md         # 本文档
```

## 🚀 启动方式

1. **使用本地服务器运行**（必需，因为需要 ES 模块和摄像头权限）

   ```bash
   # Python 3
   python -m http.server 8000
   
   # 或使用 Node.js http-server
   npx http-server -p 8000
   ```

2. **访问应用**
   - 打开浏览器访问 `http://localhost:8000`
   - 允许摄像头访问权限

## 🎮 操作说明

### 👈 左手控制（琶音基础）

1. **上下移动**：控制音高
   - 向上移动 → 更高音
   - 向下移动 → 更低音

2. **拇指与食指距离**：控制音量
   - 捏紧（距离小）→ 音量小
   - 张开（距离大）→ 音量大

3. **握拳**：切换音色预设
   - 握拳一次切换一个音色（共 8 种音色

### 👉 右手控制（欧几里得模式）

1. **拇指 + 食指捏合**：激活 XY Pad 控制
   - 捏合时：右手位置控制 XY Pad
   - X 轴（左右）：控制节奏预设（9 种节奏模式）
   - Y 轴（上下）：控制和声复杂度（向上 = 更多音符）

2. **松开**：脱离控制
   - 松开后 XY Pad 保持最后位置，继续播放当前模式

## 🎵 音色预设

从 arpeggiator-remix 保留的 8 种音色：

1. **DX7 E.PIANO 1** - 经典电钢琴
2. **DX7 BRASS 1** - 经典铜管音色
3. **DX7 MARIMBA** - 马林巴木琴
4. **Clean Sine Wave** - 纯净正弦波
5. **DX7 SYNTHWAVE LEAD** - 合成器主音
6. **DX7 CRYSTAL PLUCK** - 水晶拨弦音
7. **E-BELL SOFT** - 柔和电子钟
8. **BR LEAD 80s** - 80 年代主音

## 🎨 可视化说明

### 双层渲染设计

1. **底层**：摄像头实景画面
   - 显示你的手部动作
   - 半透明黑色覆盖层，让画面可见但不干扰

2. **上层 Canvas**：坐标平面和可视化
   - **左侧**：和声块显示（从下往上，8 个块）
     - 白色边框 = 激活的和声音符
     - 灰色边框 = 未激活
     - 闪烁效果 = 音符触发时
   
   - **主区域**：节奏可视化
     - 背景竖线 = 节奏事件位置
     - 动画条 = 音符播放进度（从左到右移动）
   
   - **十字线**：XY Pad 位置指示
     - 当右手捏合时显示
     - 交点即为当前 XY Pad 坐标

### 手部位置指示器

- **蓝色圆点**：左手位置
- **绿色圆点**：右手位置（仅当捏合时显示）

## 🔧 技术实现

### 手势识别

- 使用 **MediaPipe HandLandmarker** 进行实时手部跟踪
- 支持双手同时检测
- 平滑处理减少抖动

### 欧几里得算法

- 生成均匀分布的节奏和和声模式
- 9 种节奏预设（从稀疏到密集）
- 8 种和声复杂度级别

### 音频系统

- 使用 **Tone.js** 进行音频合成
- **PolySynth** + **FMSynth** 生成音色
- 内置混响效果

## 🐛 常见问题

### Q: 摄像头无法启动
**A:** 
- 确保浏览器有摄像头权限
- 检查是否有其他应用占用摄像头
- 尝试刷新页面重新授权

### Q: 手势识别不准确
**A:**
- 确保光线充足
- 手部与摄像头保持适当距离（50-100cm）
- 背景不要太复杂

### Q: 没有声音
**A:**
- 确保浏览器已允许音频自动播放
- 首次使用需要用户手势交互后才会启动音频
- 检查系统音量设置

### Q: 右手捏合检测不稳定
**A:**
- 可以调整 `rightHandState.pinchThreshold`（在 main.js 中）
- 默认值为 0.05，可以尝试 0.03-0.07 之间
